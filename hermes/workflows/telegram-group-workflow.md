Workflow:
1. User sends command in Telegram Group.
2. Lio receives command.
3. Lio posts status to 🧭 Operator.
4. Lio calls Cala internally.
5. Cala screening result goes to 🔎 Screening.
6. Lio sends top candidates to Konlin.
7. Konlin analysis goes to 🧠 Analyst.
8. Lio final result goes to 📊 Result.
9. System checks go to 📡 System.
10. Entry/execution messages go to 🚀 Entry.
11. Errors go to 🛠 Troubleshoot.
